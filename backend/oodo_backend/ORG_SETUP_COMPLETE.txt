🔐 SECURITY IMPLEMENTATION
═══════════════════════════════════════════════════════════════════════════

✅ Method-Level Security
   - @PreAuthorize("hasRole('ADMIN')") on all admin endpoints
   - @EnableMethodSecurity in SecurityConfig
   - Returns 403 Forbidden for non-admin users

✅ Role Promotion Enforcement
   - Role > EMPLOYEE can ONLY be assigned through UserService.promoteRole()
   - ADMIN role self-assignment architecturally impossible
   - Validation: Rejects ADMIN and EMPLOYEE role assignments
   - Code comments enforce this critical business rule

✅ Data Integrity
   - Circular hierarchy detection prevents logical data corruption
   - Soft delete for departments preserves referential integrity
   - Unique name validation prevents duplicates
   - Lazy loading for relationships prevents N+1 queries

✅ Authentication
   - JWT token required in Authorization header
   - Token validation through existing JwtAuthFilter
   - User must have ADMIN role for access

🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════

1. Start Database
   $ docker-compose up -d

2. Build Project
   $ mvn clean install

3. Run Application
   $ mvn spring-boot:run

4. Create ADMIN User
   - Signup as employee
   - Promote to ADMIN via endpoint (requires existing admin, or direct DB)

5. Test Department Creation
   $ curl -X POST http://localhost:8080/api/admin/departments \
     -H "Authorization: Bearer <admin_token>" \
     -H "Content-Type: application/json" \
     -d '{"name":"Engineering","headId":null,"parentDepartmentId":null}'

6. Test Category Creation
   $ curl -X POST http://localhost:8080/api/admin/categories \
     -H "Authorization: Bearer <admin_token>" \
     -H "Content-Type: application/json" \
     -d '{"name":"Laptops","customFields":{"warrantyPeriod":24}}'

7. Test Role Promotion
   $ curl -X PATCH http://localhost:8080/api/admin/employees/<user-id>/role \
     -H "Authorization: Bearer <admin_token>" \
     -H "Content-Type: application/json" \
     -d '{"role":"ASSET_MANAGER"}'
